# .NET Real-Time Chat App

This is an **ongoing portfolio project**: a real-time chat application built with .NET technologies.  
The goal is to learn and demonstrate skills in **real-time communication, web development, and deployment**.  

## Project Goals
- Build a full-stack chat app with **real-time messaging**.
- Practice **ASP.NET Core + SignalR** for backend communication.
- Explore **React** for the frontend UI.
- Deploy a cloud-ready application with **Docker + Azure (or AWS) + GitHub Actions**.
- Showcase a production-quality project on **GitHub**.

## Planned Tech Stack
- **Backend**: ASP.NET Core, SignalR, Entity Framework Core
- **Database**: SQL Server (Azure SQL for deployment)
- **Frontend**: React + TypeScript (SignalR JS Client)
- **Authentication**: JWT-based Auth (ASP.NET Identity)
- **Deployment**: Docker, Azure App Service
- **Testing**: xUnit, MSTest
